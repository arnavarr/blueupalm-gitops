Genera una presentación técnica con Slidev a partir del tema proporcionado.

## Instrucciones

1. Crear directorio `presentations/slidev/` si no existe
2. Generar `slides.md` con el contenido solicitado
3. Usar theme `seriph`, diagramas Mermaid, code highlighting
4. Máximo 12 slides por presentación
5. Instalar dependencias con `npm install` y arrancar con `npm run dev`

## Referencia

Consultar KI `presentation-tooling` para guía completa de Slidev.
