Genera una presentación comercial/ejecutiva con Presenton a partir del tema proporcionado.

## Instrucciones

1. Verificar que Presenton está corriendo: `docker ps --filter name=presenton`
2. Si no está corriendo, arrancar con Docker (LLM=google, GOOGLE_MODEL=models/gemini-2.5-flash)
3. Abrir interfaz web: http://localhost:5000
4. Preparar un prompt estructurado con el contexto del proyecto
5. Exportar a PPTX o PDF

## Referencia

Consultar KI `presentation-tooling` para guía completa de Presenton.
